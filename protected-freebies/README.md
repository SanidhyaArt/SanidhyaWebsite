# Protected Freebies

Put downloadable freebie files in this folder.

How to add a new freebie:

1. Drop the file here.
2. Add a matching entry in `api/_freebies-security.js` inside `freebieCatalog`.
3. Add a matching `data-freebie-file="your-freebie-id"` button in `hidden.html`.
